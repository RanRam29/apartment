# 1. סקירה כללית
## 1.1 מהי DirApp?
DirApp היא פלטפורמה ישראלית לחיבור בין משכירי דירות לשוכרים פוטנציאליים, עם ניהול מלא של תהליך השכירות מההיכרות הראשונה ועד לסיום החוזה.
המוצר כולל שני ממשקים: אפליקציה מובייל (React Native / Expo) לשוכרים ומשכירים, ודף web נפרד לערבים. פאנל Admin לניהול המערכת מלא (GODMODE).
## 1.2 ערכי ליבה
ניטרליות מוחלטת — DirApp אינה צד בכל סכסוך בין משתמשים
פרטיות by design — שמירת מידע מינימלית לפי תיקון 13 לחוק הגנת הפרטיות
קינפוגביליות מלאה — כל פרמטר עסקי ניתן לשינוי מפאנל Admin
חוק ישראלי — תנאי שימוש ופעילות כפופים לדין הישראלי
